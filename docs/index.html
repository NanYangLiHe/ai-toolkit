What Can You Build?

📝

Auto Writer

Generate articles, documentation, and translations using your local LLM. No API limits, no data leakage.

python auto_writer.py --topic "Your Topic"
🔍

Code Review

Automated code analysis and improvement suggestions. Catch bugs before they ship!

python code_review.py --file script.py
📊

Data Parser

Extract structured JSON/CSV from messy text. Perfect for invoices, contacts, meeting notes.

python data_parser.py --input notes.txt --format json
🌐

Web Scraper

Intelligent web scraping with AI-powered parsing. Extract exactly what you need.

python web_scraper.py --url https://example.com

Setup in 3 Minutes

# Clone the repository
git clone https://github.com/NanYangLiHe/ai-toolkit.git

# Install dependencies
pip install -r requirements.txt

# Configure your local LLM endpoint
cp .env.example .env
# Edit .env with your settings (Ollama/LM Studio/vLLM)

# Start using the tools!
python scripts/auto_writer.py --topic "Hello World"
